Version 1.0.1 May 10 2025

Major Performance BoostEnhancement

Upgraded to .NET 9, delivering 15-25% performance improvements across the entire application for faster data processing and smoother user interactions. It also uses 20-30% less memory.

Optimized Chart ManagementEnhancement

Separated Totals_Chart and Distribution_Chart for purchases and sales, eliminating unnecessary refreshes when switching between screens and significantly boosting performance.

Enhanced language switching with a new loading screen and optimized the translation system. It downloads all translations at once instead of using API calls, making language conversion much faster and more reliable.

Enhanced Chart InteractivityNew Feature

Added an option to reset the zoom of charts and improved chart zooming capabilities for a better user experience when analyzing data visualizations.

Connectivity ManagementNew Feature

Added internet connection monitoring that displays user-friendly messages when connectivity is unavailable, replacing confusing error messages.

Enhanced Loading ControlsNew Feature

Added a Cancel button to some loading screens, giving users more control and the ability to interrupt long-running operations when needed.
